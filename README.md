# Página de Cadastro
Projeto desenvolvido para o processo seletivo da Docket.\
Foram produzidas duas versões diferentes, cada qual em sua própria branch:
- [Versão sem framework](https://github.com/miguel-malagoli/pagina-de-cadastro)
- [Versão com React (esta branch)](https://github.com/miguel-malagoli/pagina-de-cadastro/tree/react)

A versão sem framework está disponível no Github Pages, [neste link](https://miguel-malagoli.github.io/pagina-de-cadastro)

## Ferramentas
Na versão com React foram utilizados, além do próprio framework, Typescript, Axios, Sass e Autoprefixer.\
Apenas HTML5, CSS3 e JavaScript puros foram utilizados na versão sem framework.
